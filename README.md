# nptools

Illicit automations for neopets.

## Quick start

Put

```
NP_USER='username'
NP_PASS='password'
```

in a file called .env in this directory. Then run `./run.sh daemon.py`.

Additional flags include `USER_AGENT` and `FIREFOX_COOKIES_DB`.
